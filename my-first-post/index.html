arch Installation-guide

Begin

Verify the boot mode

To verify the boot mode, list the efibars directory

ls /sys/firmware/efi/efivars

Connect to the internet

For wireless, make sure the wireless card is not blocked with rfkill

Many laptops have a hardware button (or switch) to turn off wireless card, however, the card can also be blocked by kernel
This can be handled by rfkill. To show the current status

rfkill list

If the card is hard-blocked, use the hardware button (switch) to unblock it
If the card is not hard-blocked but soft-blocked, use the following command

rfkill unblock wifi

Then use iwctl to connect internet

iwctl

will enter a shell environment

device list

Then do

station wlan0 scan
station wlan0 get-networks
station wlan0 connect Tomorrow

at last

quit

Update the system clock

Use timedatectl to ensure the system clock is accurate

timedatectl set-ntp true

Partition the disks

When recognized by the live system, disks are assigned to a block device such as /dev/sda, /dev/nvme0n1 or /dev/mmcblk0
To identify these devices, use lsblk or fdisk

fdisk -l

Mount partition

mount /dev/nvme0n1p3 /mnt

if EFI/GPT

mkdir /mnt/boot
mount /dev/nvme0n1p1 /mnt/boot

Set mirrors

vim /etc/pacman.d/mirrorlist

Aliyun

Server = http://mirrors.aliyun.com/archlinux/$repo/os/$arch

Install base package

pacstrap /mnt base base-devel linux linux-firmware dhcpcd

configure Fstab

genfstab -L /mnt >> /mnt/etc/fstab

Chroot

arch-chroot /mnt

Environment

Put environment.sh in /etc/profile.d

Network

Install NetworkManager

sudo pacman -S networkmanager network-manager-applet iwd dhcpcd

NetworkManager defaults to use wpa_supplicant as back-end
iwd has better performance and less resource consumption
change to iwd needs edit the configuration

sudo vim /etc/NetworkManager/conf.d/wifi_backend.conf

add

[device]
wifi.backend=iwd

and reboot
